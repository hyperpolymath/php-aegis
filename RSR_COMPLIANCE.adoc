== File Format Notes

=== Platform-Required Formats

These files use Markdown (`.md`) because GitHub/GitLab automation specifically looks for them:

* `CODE_OF_CONDUCT.md` - GitHub Community Standards detection
* `SECURITY.md` - GitHub Security tab integration
* `CONTRIBUTING.md` - GitHub contribution guidelines
* `FUNDING.yml` - GitHub Sponsors integration (if present)

=== RSR Formats

These files use RSR-preferred formats:

* `README.adoc` - AsciiDoc for rich documentation
* `LICENSE.txt` - Plain text for universal compatibility
* `STATE.scm` - Scheme for machine-readable project state
* `CITATION.cff` - Citation File Format standard
